Upvote/Downvote Images with TheCatAPI

This is an interactive webpage where the user will select a cat breed, then hit a button to display an image of that selected breed. The user will then be able to hit an upvote/downvote button if they like or disliked the image shown. This project makes use of FetchAPI, GET/POST, asynchronous functions, and DOM to get values and create new HTML elements.
